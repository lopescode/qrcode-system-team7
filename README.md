## Installation

```bash
$ npm install
```

## Running the app

```bash
# development mode
$ npm run dev:web
$ npm run dev:server

# production mode
$ npm run prod:server
```

## Running the app with Docker

```bash
# development mode
$ docker-compose up

# production mode
$ docker-compose -f docker-compose.prod.yml up
```

## Running migrations

```bash
# development mode
$ npm run migrate:dev

# production mode
$ npm run migrate:prod
```
